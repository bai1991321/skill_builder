The instruction is the following....
1. you have to install mysql server on your computer.
- install XAMPP or MYSQL server
- import skill_builder_db inside query folder on MySQL Database.
2. Run backend server.
- go to backend folder and run 'npm install' and 'nodemon app'
if you didn't install nodemon cli on your computer, run the following
npm install -g nodemon

3. Run Frontend app.
- go to frontend folder 
- npm install 
- npm install --save node-sass
- ng serve --open

then our app will be run in your browser.
